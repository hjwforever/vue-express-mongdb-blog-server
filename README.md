# vue-express-mongdb-blog-server
vue-express-mongdb-blog文章管理系统的后端系统，
带有登录注册及相关中间件、权限区别、分页搜索、增删改查文章功能等api接口;
## 前端Vue，后端Express，数据库Mongodb 

## Project setup
```
npm install
```

## Run Server
```
node index.js
```
正常运行后即可访问 http://localhost:8080 访问页面
